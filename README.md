# LogLog

[![CircleCI](https://circleci.com/gh/wawawatataru/loglog.svg?style=svg)](https://circleci.com/gh/wawawatataru/loglog)

## Lint&Test

下記の lint とテストが実装されます。

```
$ ./bin/lintest
```

## Lint

rubocop と slim-lint が実行されます。

```
$ ./bin/lint
```

## Test

model のテストと system テストが実行されます。

```
$ ./bin/test
```
